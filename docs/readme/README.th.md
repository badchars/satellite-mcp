<h3 align="center">เซิร์ฟเวอร์ MCP สำหรับข่าวกรองภูมิสารสนเทศแบบครบวงจรสำหรับเอเจนต์ AI</h3>

<p align="center">
  Sentinel-2, Landsat, NASA FIRMS, แสงสว่างยามค่ำคืน, การติดตามอากาศยานและเรือ, ข่าวกรองทางทหารและการป้องกันประเทศ, เหตุการณ์ความขัดแย้ง, การคว่ำบาตร, ภูมิศาสตร์ไซเบอร์, ข้อมูลสภาพอากาศ &mdash; รวมอยู่ในเซิร์ฟเวอร์ MCP เดียวพร้อม 171 เครื่องมือใน 27 หมวดหมู่<br>
  เอเจนต์ AI ของคุณจะได้รับ<b>ข่าวกรองภูมิสารสนเทศแบบครบวงจรตามต้องการ</b> ไม่ใช่ API หลายสิบตัวและการเชื่อมโยงข้อมูลด้วยมือ

## ภาพรวม

**satellite-mcp** เป็นเซิร์ฟเวอร์ MCP สำหรับข่าวกรองภูมิสารสนเทศแบบครบวงจร (GEOINT) ที่มอบเครื่องมือ 171 รายการใน 27 หมวดหมู่ให้กับเอเจนต์ AI ของคุณผ่าน [Model Context Protocol](https://modelcontextprotocol.io) ตั้งแต่ภาพถ่ายดาวเทียมและการตรวจจับไฟป่า ไปจนถึงการติดตามอากาศยาน การวิเคราะห์การจราจรทางเรือ ข่าวกรองทางทหาร และการเฝ้าระวังสถานการณ์ความขัดแย้ง &mdash; เอเจนต์สามารถเข้าถึงทุกอย่างได้ในการสนทนาเดียว

---

## เริ่มต้นอย่างรวดเร็ว

### ตัวเลือกที่ 1: npx (ไม่ต้องติดตั้ง)

```bash
npx satellite-mcp
```

### ตัวเลือกที่ 2: ติดตั้งแบบ Global

```bash
npm install -g satellite-mcp
satellite-mcp
```

### เชื่อมต่อกับเอเจนต์ AI ของคุณ

<details open>
<summary><b>Claude Code</b></summary>

```bash
claude mcp add satellite-mcp -- npx satellite-mcp
```

</details>

<details>
<summary><b>Claude Desktop</b></summary>

เพิ่มในไฟล์ `~/Library/Application Support/Claude/claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "satellite": {
      "command": "npx",
      "args": ["-y", "satellite-mcp"]
    }
  }
}
```

</details>

<details>
<summary><b>Cursor / Windsurf / ไคลเอนต์ MCP อื่นๆ</b></summary>

ใช้รูปแบบการตั้งค่า JSON เดียวกัน ชี้คำสั่งไปที่ `npx satellite-mcp` หรือพาธการติดตั้งในเครื่อง

</details>

## หมวดหมู่เครื่องมือ (27 หมวดหมู่ / 171 เครื่องมือ)

| หมวดหมู่ | เครื่องมือ | คำอธิบาย |
|----------|-----------|---------|
| Sentinel-2 | 8 | ภาพถ่ายดาวเทียมเชิงแสงจาก ESA Sentinel-2 |
| Landsat | 7 | ภาพถ่ายหลายช่วงคลื่นจากโปรแกรม USGS Landsat |
| NASA FIRMS | 6 | ไฟป่าที่กำลังลุกไหม้และความผิดปกติทางความร้อนแบบเกือบเรียลไทม์ |
| NASA Earth | 7 | ข้อมูลการสังเกตการณ์โลกและเหตุการณ์ทางธรรมชาติจาก NASA |
| แสงสว่างยามค่ำคืน | 5 | ข้อมูลแสงสว่างยามค่ำคืน VIIRS สำหรับการขยายตัวของเมืองและกิจกรรมทางเศรษฐกิจ |
| Planet Labs | 4 | ภาพถ่ายรายวันความละเอียดสูงจากกลุ่มดาวเทียม Planet Labs |
| ข่าวกรองการบิน | 10 | การติดตามอากาศยาน การระบุตัวตน และข่าวกรองเส้นทางบิน |
| ข่าวกรองทางทะเล | 8 | การติดตามเรือ กิจกรรมท่าเรือ และการรับรู้สถานการณ์ทางทะเล |
| ทหารและการป้องกัน | 8 | สิ่งอำนวยความสะดวกด้านกลาโหม การส่งกำลังบำรุง และการเฝ้าระวังกิจกรรมทางทหาร |
| อวกาศและวงโคจร | 6 | การติดตามดาวเทียม สภาพอากาศในอวกาศ และข่าวกรองวงโคจร |
| ความขัดแย้งและเหตุการณ์ | 6 | เขตความขัดแย้ง ข้อมูลเหตุการณ์ ACLED และการติดตามความรุนแรง |
| สิ่งแวดล้อม | 6 | สภาพภูมิอากาศ คุณภาพอากาศ การสูญเสียพื้นที่ป่า และการเฝ้าระวังสิ่งแวดล้อม |
| โครงสร้างพื้นฐานสำคัญ | 6 | พลังงาน การขนส่ง และการปกป้องโครงสร้างพื้นฐานสำคัญ |
| การคว่ำบาตร | 5 | รายชื่อการคว่ำบาตรระหว่างประเทศ การปฏิบัติตามกฎระเบียบการค้า และการตรวจสอบ |
| ภูมิประเทศ | 6 | ข้อมูลระดับความสูง การวิเคราะห์ภูมิประเทศ และข่าวกรองภูมิศาสตร์ |
| ภูมิศาสตร์ไซเบอร์ | 6 | การระบุตำแหน่งทางภูมิศาสตร์ของภัยคุกคามทางไซเบอร์และ IP |
| ประชากร | 5 | ข้อมูลประชากรศาสตร์และการวิเคราะห์ความหนาแน่นของประชากร |
| เกษตรกรรม | 5 | สุขภาพพืชผล ดัชนีพืชพรรณ และการเฝ้าระวังทางการเกษตร |
| มนุษยธรรม | 5 | การทำแผนที่วิกฤต กระแสผู้ลี้ภัย และการตอบสนองด้านมนุษยธรรม |
| มหาสมุทร | 5 | อุณหภูมิผิวน้ำทะเล กระแสน้ำ และการเฝ้าระวังมหาสมุทร |
| การค้า | 5 | การค้าระหว่างประเทศ เส้นทางการขนส่ง และข่าวกรองห่วงโซ่อุปทาน |
| แผ่นดินไหว | 5 | การเฝ้าระวังแผ่นดินไหว กิจกรรมทางธรณีวิทยา และการเฝ้าระวังเปลือกโลก |
| OSM | 8 | ข้อมูล OpenStreetMap การค้นหาโครงสร้างพื้นฐาน และการสืบค้นเชิงพื้นที่ |
| การแปลงพิกัด | 8 | การค้นหาที่อยู่ การแปลงพิกัดย้อนกลับ และการค้นหาสถานที่ |
| การวิเคราะห์สเปกตรัม | 8 | การวิเคราะห์ภาพหลายช่วงคลื่นและไฮเปอร์สเปกตรัม |
| การตรวจจับการเปลี่ยนแปลง | 6 | การเปรียบเทียบตามเวลาและการตรวจจับการเปลี่ยนแปลงในภาพดาวเทียม |
| สภาพอากาศ | 6 | สภาพอากาศปัจจุบัน การพยากรณ์ และข้อมูลอุตุนิยมวิทยา |

## คีย์ API

เครื่องมือหลายตัวทำงานได้โดยไม่ต้องใช้คีย์ API โดยอาศัยแหล่งข้อมูลฟรี แหล่งข้อมูลพรีเมียมต้องใช้คีย์ API แยกต่างหากที่สามารถตั้งค่าเป็นตัวแปรสภาพแวดล้อมได้

```bash
export SENTINEL_API_KEY=your-key
export NASA_API_KEY=your-key
export PLANET_API_KEY=your-key
export OPENWEATHER_API_KEY=your-key
```

คีย์ทั้งหมดเป็นตัวเลือก เซิร์ฟเวอร์จะลดระดับการทำงานอย่างสง่างามและส่งข้อความแสดงข้อผิดพลาดที่อธิบายได้ชัดเจนเมื่อไม่มีคีย์สำหรับเครื่องมือที่กำหนด

## สถาปัตยกรรม

```
src/
  index.ts                # จุดเริ่มต้น CLI และการตั้งค่าเซิร์ฟเวอร์ MCP
  protocol/
    tools.ts              # ทะเบียนเครื่องมือ — เครื่องมือทั้ง 171 รายการลงทะเบียนที่นี่
  providers/
    sentinel/             # เครื่องมือภาพถ่าย Sentinel-2 (8)
    landsat/              # เครื่องมือภาพถ่าย Landsat (7)
    firms/                # เครื่องมือไฟป่า NASA FIRMS (6)
    nasa-earth/           # การสังเกตการณ์โลกจาก NASA (7)
    nightlights/          # เครื่องมือแสงสว่างยามค่ำคืน VIIRS (5)
    planet/               # เครื่องมือ Planet Labs (4)
    aircraft/             # เครื่องมือข่าวกรองการบิน (10)
    maritime/             # เครื่องมือข่าวกรองทางทะเล (8)
    military/             # เครื่องมือทหารและการป้องกัน (8)
    space/                # เครื่องมืออวกาศและวงโคจร (6)
    conflict/             # เครื่องมือความขัดแย้งและเหตุการณ์ (6)
    environmental/        # เครื่องมือสิ่งแวดล้อม (6)
    infrastructure/       # เครื่องมือโครงสร้างพื้นฐานสำคัญ (6)
    sanctions/            # เครื่องมือการคว่ำบาตร (5)
    terrain/              # เครื่องมือภูมิประเทศ (6)
    cyber-geo/            # เครื่องมือภูมิศาสตร์ไซเบอร์ (6)
    population/           # เครื่องมือประชากร (5)
    agriculture/          # เครื่องมือเกษตรกรรม (5)
    humanitarian/         # เครื่องมือมนุษยธรรม (5)
    ocean/                # เครื่องมือมหาสมุทร (5)
    trade/                # เครื่องมือการค้า (5)
    seismic/              # เครื่องมือแผ่นดินไหว (5)
    osm/                  # เครื่องมือ OpenStreetMap (8)
    geocoding/            # เครื่องมือการแปลงพิกัด (8)
    spectral/             # เครื่องมือการวิเคราะห์สเปกตรัม (8)
    change-detection/     # เครื่องมือการตรวจจับการเปลี่ยนแปลง (6)
    weather/              # เครื่องมือสภาพอากาศ (6)
```

**หลักการออกแบบ:**

- **27 ผู้ให้บริการ, 1 เซิร์ฟเวอร์** &mdash; แหล่งข้อมูลแต่ละแห่งเป็นโมดูลอิสระ เอเจนต์จะเลือกเครื่องมือที่เหมาะสมตามคำถาม
- **ตัวจำกัดอัตราต่อผู้ให้บริการ** &mdash; แหล่งข้อมูลแต่ละแห่งมี `RateLimiter` ของตัวเองที่ปรับเทียบตามขีดจำกัดของ API นั้นๆ
- **แคชแบบ TTL** &mdash; ข้อมูลดาวเทียม สภาพอากาศ และฟีดภูมิรัฐศาสตร์ถูกแคชเพื่อหลีกเลี่ยงการเรียก API ที่ซ้ำซ้อน
- **การลดระดับอย่างสง่างาม** &mdash; คีย์ API ที่ขาดหายไปจะไม่ทำให้เซิร์ฟเวอร์ล่ม เครื่องมือจะส่งข้อความแสดงข้อผิดพลาดที่อธิบายได้ชัดเจน
- **ไม่ต้องตั้งค่า** &mdash; `npx satellite-mcp` &mdash; คำสั่งเดียว ไม่ต้องตั้งค่าใดๆ
